# LitePad

[中文](./README.md)

A scratchpad for quick notes. Hotkey to summon, use and go.

## Features

- Multi-tab management
- Global hotkey (Alt+X)
- Inline expression calculation (`1+1=` then press Ctrl+Enter)
- Window position and size memory
- Custom shortcuts

![LitePad Preview](assets/flashpad-reference.png)

## Development

```bash
npm install
npm run dev        # dev
npm run build:win  # build
```
