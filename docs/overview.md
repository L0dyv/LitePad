# FlashPadSelf - 项目总览

## 项目背景

现有的 FlashPad 软件存在一些编辑体验问题（如 Ctrl+Backspace 行为异常），因此决定自行开发一个类似的快速记事本工具。

## 核心目标

1. **极速响应** - 全局快捷键（Alt+X）toggle，毫秒级响应
2. **优秀编辑体验** - 正常的文本编辑行为（Ctrl+Backspace、选词等）
3. **本地离线** - 数据存储在本地，无网络依赖
4. **计算功能** - 输入表达式后按 Ctrl+Enter 计算结果

## 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | **Electron** | 托盘常驻 + 窗口 hide/show 实现即时响应 |
| 前端 | **React** | 组件化开发 |
| 编辑器 | **CodeMirror 6** | 轻量、现代、高度可定制 |
| 数据存储 | **SQLite** 或 **JSON 文件** | 多笔记管理 |
| 样式 | **CSS / Tailwind** | 待定 |

### 编辑器选型说明

**CodeMirror 6** vs **Monaco Editor**：

| 特性 | CodeMirror 6 | Monaco Editor |
|------|--------------|---------------|
| 体积 | ~150KB | ~2MB+ |
| 性能 | 更轻量 | 较重 |
| 框架兼容 | React/Vue/原生均可 | 同样兼容 |
| 定制性 | 极高（模块化设计）| 高 |
| 适用场景 | 轻量编辑器 | 完整 IDE 体验 |

**结论**：对于速效记事本场景，CodeMirror 6 更合适。

> 注：编辑器与框架无强绑定关系，CodeMirror 6 和 Monaco 都有官方或社区的 React 封装。
