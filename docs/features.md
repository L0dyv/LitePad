# 功能需求

## MVP（最小可用版本）

### P0 - 必须实现

- [ ] **全局快捷键 Toggle**
  - 默认 Alt+X（可配置）
  - 托盘常驻，窗口 show/hide（非销毁重建）
  - 响应时间 < 100ms

- [ ] **多标签页管理**
  - 类似 FlashPad 的标签页切换
  - 新建/关闭/重命名标签页
  - 标签页顺序可拖拽调整

- [ ] **自动保存**
  - 无需手动保存，实时持久化
  - 数据存储在本地（SQLite/JSON）

- [ ] **基础文本编辑**
  - 正常的剪切/复制/粘贴
  - Ctrl+Backspace 删除前一个单词
  - Ctrl+Z/Y 撤销/重做
  - 查找/替换（Ctrl+F / Ctrl+H）

- [ ] **计算功能**
  - 输入 `2+3*4=` 后按 Ctrl+Enter
  - 自动计算并输出结果：`2+3*4=14`
  - 支持基本四则运算、括号、幂运算

### P1 - 重要功能

- [ ] **另存为**
  - 当前标签页内容导出为 .txt/.md 文件

- [ ] **系统托盘**
  - 右键菜单：显示/隐藏、设置、退出
  - 托盘图标点击 toggle 窗口

- [ ] **窗口行为设置**
  - 固定位置 / 居中 / 记住上次位置
  - Always on Top（可选）

### P2 - 优化体验

- [ ] **主题切换**
  - 浅色 / 深色模式
  - 跟随系统

- [ ] **自定义快捷键**
  - 用户可修改全局 toggle 快捷键

- [ ] **搜索历史笔记**
  - 跨标签页全文搜索
